# TrashDay

サイトにアクセスすると、今日回収されるごみの種類を表示します  
忘れがちな「今日が何ゴミの日か」が一目でわかります  

## 主な機能

* 今日収集されるごみの種類を表示
  * 今週の月～日曜日の日付とごみの種類が表示される
  * 回収時刻（期限）を表示する
  * 回収場所を表示する
* 可燃ごみ、不燃ごみ、資源ごみなどの区別
* 不定期の回収日が表示される
* 指定した地域の収集ルールに対応（将来的にできれば）
* 明日以降の収集予定の表示（将来的にできれば）
* メッセージアプリへの通知（将来的にできれば）

## インストール手順

リポジトリをクローン

```bash
git clone ---
```

## 使い方

仮想環境を有効化した状態で、 `main.py` を実行
