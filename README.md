# TrashDay

今日回収されるごみの種類を表示します  
今日が何ゴミの日か忘れがちなので作りました  

## 主な機能

* 本日収集されるごみの種類を表示
* 可燃ごみ、不燃ごみ、資源ごみなどの区別
* 指定した地域の収集ルールに対応（将来的にできれば）
* 明日以降の収集予定の表示（将来的にできれば）
* メッセージアプリへの通知（将来的にできれば）

## インストール手順

リポジトリをクローン

```bash
git clone ---
```

## 使い方

仮想環境を有効化した状態で、 `main.py` を実行

```bash
python main.py
```
